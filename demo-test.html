Demo Flow

  1. 用户点击"Demo"按钮
  2. 系统进入Demo模式，显示"Demo starting..."
  3. 喝水提醒自动开始（30秒倒计时）
  4. 10秒后站立提醒自动开始（30秒倒计时）
  5. 用户会看到两个浏览器通知依次弹出
  6. Demo自动结束并恢复正常状态

Technical Features

  • MVP原则：最小可行产品，专注核心功能
  • 无侵入性：不影响正常应用功能
  • 自动清理：Demo结束后完全恢复状态
  • 错误处理：包含基本的错误处理机制
  • 响应式设计：支持移动端和桌面端
  • 用户友好：清晰的状态提示和反馈
